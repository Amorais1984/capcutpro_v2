## 🎯 Visão Geral

O **CapCut SRT Pro** é um conversor avançado de legendas especialmente otimizado para o CapCut. Ele transforma seus roteiros em arquivos de legenda profissionais com formatação inteligente e timing perfeito.

### ✨ Principais Benefícios:
- **Formatação Automática**: Quebra inteligente de texto respeitando pontuação
- **Timing Otimizado**: Presets específicos para diferentes tipos de conteúdo
- **Múltiplos Formatos**: SRT, VTT, ASS e ZIP
- **Preview em Tempo Real**: Veja como ficará antes de exportar
- **Interface Moderna**: Design responsivo e intuitivo

---

## 🚀 Como Usar

### Passo 1: Inserir o Texto
Você tem duas opções para inserir seu roteiro:

**Opção A - Importar Arquivos:**
1. Clique em "Escolher Arquivos" no painel "📝 Entrada de Texto"
2. Selecione um ou múltiplos arquivos `.txt`
3. Os arquivos serão automaticamente combinados em ordem alfabética

**Opção B - Colar Texto:**
1. Cole seu roteiro diretamente na área de texto
2. O texto será analisado automaticamente

### Passo 2: Configurar Timing
No painel "⚙️ Configurações de Timing":

**Presets Rápidos:**
- **Rápido (3s)**: Para conteúdo dinâmico, vlogs, tutoriais rápidos
- **Normal (5s)**: Padrão recomendado para a maioria dos vídeos
- **Lento (8s)**: Para conteúdo educativo, explicações detalhadas
- **Auto**: Calcula automaticamente baseado na velocidade de leitura

**Configuração Manual:**
- **Duração por bloco**: Tempo que cada legenda ficará na tela (1-30s)
- **Intervalo entre blocos**: Pausa entre legendas (0-10s)
- **Máx. caracteres por linha**: Limite de caracteres (20-50)
- **Máx. linhas por bloco**: Quantas linhas por legenda (1-3)

### Passo 3: Converter
1. Clique em "🚀 Converter para SRT"
2. Aguarde a barra de progresso completar
3. Visualize o resultado no painel de preview

### Passo 4: Exportar
Escolha o formato desejado:
- **📄 Download SRT**: Formato padrão do CapCut
- **🎞️ Download VTT**: Para players web
- **🎨 Download ASS**: Com estilos avançados
- **📦 Download ZIP**: Múltiplos formatos

## 📊 Funcionalidades Principais

### Estatísticas em Tempo Real
O painel mostra automaticamente:
- **Palavras**: Contagem total de palavras
- **Caracteres**: Número de caracteres no texto
- **Blocos**: Quantidade estimada de legendas
- **Duração**: Tempo total do vídeo com legendas

### Formatação Inteligente
- **Quebra por Pontuação**: Respeita pontos, vírgulas e exclamações
- **Limite de Caracteres**: Evita legendas muito longas
- **Divisão Natural**: Quebra em pontos lógicos da frase
- **Múltiplas Linhas**: Suporte a legendas de 1-3 linhas

### Preview Avançado
- **Visualização Imediata**: Veja o SRT formatado instantaneamente
- **Estilo Terminal**: Interface familiar para desenvolvedores
- **Scroll Automático**: Navegação fácil em textos longos
- **Numeração**: Blocos numerados como no arquivo final

## ⚙️ Configurações Avançadas

### Timing Personalizado

**Duração por Bloco:**
- **1-3 segundos**: Para textos muito curtos ou efeitos rápidos
- **3-5 segundos**: Ideal para a maioria dos conteúdos
- **5-8 segundos**: Para textos mais longos ou explicações
- **8+ segundos**: Para leitura confortável de textos complexos

**Intervalo entre Blocos:**
- **0 segundos**: Legendas contínuas (não recomendado)
- **1-2 segundos**: Transição rápida
- **2-3 segundos**: Padrão recomendado
- **3+ segundos**: Para dar tempo de processar a informação

### Formatação de Texto

**Caracteres por Linha:**
- **20-30**: Para telas pequenas (mobile)
- **30-40**: Padrão recomendado (desktop)
- **40-50**: Para textos técnicos ou telas grandes

**Linhas por Bloco:**
- **1 linha**: Legendas minimalistas
- **2 linhas**: Padrão recomendado
- **3 linhas**: Para textos mais densos (use com moderação)

## 📤 Formatos de Exportação

### SRT (SubRip)
- **Uso**: Formato padrão do CapCut
- **Compatibilidade**: Universal
- **Características**: Simples, leve, amplamente suportado

### VTT (WebVTT)
- **Uso**: Players web, YouTube, Vimeo
- **Compatibilidade**: Navegadores modernos
- **Características**: Suporte a estilos CSS

### ASS (Advanced SubStation Alpha)
- **Uso**: Edição avançada, anime, efeitos especiais
- **Compatibilidade**: Players especializados
- **Características**: Estilos, cores, posicionamento

### ZIP (Múltiplos Formatos)
- **Uso**: Backup completo
- **Compatibilidade**: Todos os formatos incluídos
- **Características**: Conveniência para distribuição

## 🎬 Dicas para CapCut

### Importação no CapCut
1. Abra seu projeto no CapCut
2. Vá em "Texto" → "Legendas Automáticas"
3. Clique em "Importar SRT"
4. Selecione o arquivo baixado
5. Ajuste posição e estilo conforme necessário

### Configurações Recomendadas
- **Duração**: 4-6 segundos para vídeos normais
- **Intervalo**: 1-2 segundos entre legendas
- **Caracteres**: 35 por linha para melhor legibilidade
- **Linhas**: 2 linhas máximo para não ocupar muito espaço

### Estilos no CapCut
- **Fonte**: Arial ou Roboto para melhor legibilidade
- **Tamanho**: 24-32pt dependendo da resolução
- **Cor**: Branco com contorno preto
- **Posição**: Terço inferior da tela

### Sincronização
- Use o preview para verificar timing
- Ajuste a duração conforme a velocidade da fala
- Considere pausas naturais na narração
- Teste em diferentes dispositivos

## 🔧 Solução de Problemas

### Problemas Comuns

**Legendas muito rápidas:**
- Aumente a "Duração por bloco"
- Use preset "Lento" ou "Normal"
- Reduza o número de caracteres por linha

**Legendas muito longas:**
- Diminua "Máx. caracteres por linha"
- Use "Máx. linhas por bloco" = 2
- Revise o texto para ser mais conciso

**Quebras estranhas:**
- Verifique a pontuação do texto original
- Ajuste "Máx. caracteres por linha"
- Use frases mais curtas

**Arquivo não importa no CapCut:**
- Certifique-se de usar formato SRT
- Verifique se o arquivo não está corrompido
- Tente recriar o arquivo

### Dicas de Performance

**Para textos longos:**
- Divida em múltiplos arquivos menores
- Use configurações conservadoras
- Teste com uma pequena amostra primeiro

**Para melhor qualidade:**
- Revise o texto antes de converter
- Use pontuação adequada
- Evite abreviações desnecessárias
- Mantenha frases claras e diretas
